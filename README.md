# TodoList_42

一个简洁美观的待办事项列表网页应用。

## 功能特性

*   ✅ 添加新的待办任务
*   ✅ 标记任务为已完成
*   ✅ 删除任务
*   💾 使用浏览器本地存储，刷新页面数据不丢失

## 如何使用

1.  克隆本项目到本地：git clone https://github.com/enenen124/TodoList_42.git
2.  使用浏览器打开 `index.html` 文件即可开始使用。

## 项目结构
